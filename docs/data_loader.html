class data_loader.DataLoader(file_path, normalize=True)[source]¶

Bases: object

The DataLoader class is responsible for loading, processing, and normalizing wine-related datasets.

This class handles various file formats, including .npy and .xlsx, and provides methods to preprocess the data, normalize it, and prepare it for further analysis. The DataLoader class is designed to streamline the data preparation process, ensuring that the data is in the correct format and standardized for consistent use in analysis and modeling.
