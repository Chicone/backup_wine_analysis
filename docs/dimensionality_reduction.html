dimensionality_reduction module¶

class dimensionality_reduction.DimensionalityReducer(data)[source]¶

Bases: object

The DimensionalityReducer class provides tools for performing dimensionality reduction on datasets.

This class offers various methods for reducing the dimensionality of data, making it easier to visualize and analyze complex datasets. It supports popular techniques such as PCA (Principal Component Analysis), t-SNE, and UMAP, and includes functionality for cumulative variance analysis and cross-validation of PCA for classification tasks. The class is designed to handle high-dimensional data efficiently and helps in identifying the most important features or components.
